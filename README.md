# Mesosphere DCOS

* In this repository you will find information regarding Mesosphere DCOS. 
* The [Technical Report](https://github.com/vinay92/Mesosphere/blob/master/Report.md) has high level overview regarding the functionality and architecture of the DCOS. 
* The [Setup file](https://github.com/vinay92/Mesosphere/blob/master/Setup.md) has instructions on how to setup the cluster and deploy a sample application on this cluster. 
* The [Screencast](https://www.youtube.com/watch?v=sx8IkZK3e-U) contains a video which has a detailed demonstration of each step of the process while installing and deploying the application along with instructions to install cassandra and deploy the oinker app.
